Skill Badges API

Generate badge grids for your skills and technologies

Usage

GET /api/s?c=python,javascript,react&perline=3

Parameters

  • c - Comma-separated technology names
  • perline - Number of badges per line (default: 4)

Endpoints

  • /api/s - Static badges
  • /api/d - Dynamic badges
  • /api/list - List all available badges
